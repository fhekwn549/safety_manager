Runtime advisory metadata represents active or recently active platform risks as defensive evidence workflows.

- Advisory id and public tracking id.
- Affected domains.
- Evidence requirements.
- Mitigation evidence.
- Hard triggers.
- Missing evidence prompts.

Advisory content must not include exploit code, exploit commands, payloads, or reproduction steps.
